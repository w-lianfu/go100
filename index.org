***** 字符串格式化

| 功能 | 解释                                   |
|------+----------------------------------------|
| %v   | 按本值输出                             |
| %+v  | 在%v基础上，对结构体字段名和值进行展开 |
| %#v  | 输出Go语言语法格式的值                 |
| %T   | 输出Go语言语法格式的类型和值           |
| %%   | 输出 % 本体                            |
| %b   | 整型以二进制方式显示                   |
| %o   | 整型以八进制方式显示                   |
| %d   | 整型以十进制方式显示                   |
| %x   | 整型以十六进制方式显示                 |
| %X   | 整型以十六进制、字母大写方式显示       |
| %U   | Unicode 字符                           |
| %f   | 浮点数                                 |
| %p   | 指针，十六进制方式显示                 |
